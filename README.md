## 緣起

小氣鬼因為本身很小氣，電影鮮少在首輪就去觀看，

總是將片名記在心中，等著進二輪片單或上串流時再來欣賞。

但...人的記性總是不好，常常忘記這些感興趣的電影，等到有閒暇之餘時，落入無片可看的窘境。

於是這支 app 就開始進入開發階段。
